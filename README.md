# Consultar crédito

## Descrição

Este projeto foi desenvolvido para a disciplina de ABEX VI, com objetivo de criar um sistema de consulta de crédito para verificar a confiabilidade de um cliente.

## Tecnologias

- Python
- React

## Instalação

### Frontend

```bash
cd client
npm install
npm run build
npm start
```

### Backend

```bash
cd server
pip install -r requirements.txt
python app.py
```
